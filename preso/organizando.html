Shoc-ed Nim

import tables, ../project

var projectList = initTable[string, Project]()

proc addProject *( id: string ) =
  projectList[id] = Project( id: id )

proc getProject * ( id: string): Project =
  return projectList[id]
En este caso, el asterisco indica que son públicos, lo que no sucede en el caso de projectList. También se usa var, que indica que es mutable, frente a let, que se usa para inmutables
